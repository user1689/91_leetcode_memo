## 思路
hashTable, bruteForce

## python3
```python3
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:

        # time n
        # space n
        # 思路二
        # hashTable

        dic = {}
        for idx, num in enumerate(nums):
            if (target - num) in dic:
                return [dic[target - num], idx]
            dic[num] = idx
```
