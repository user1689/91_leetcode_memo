## 题目
https://www.acwing.com/problem/content/description/870/

## 思路
```python3
N = 1000010
st = [False] * N
primes = [0] * N
cnt = 0

def main():
    global cnt
    x = int(input())
    for i in range(2, x+1):
        if (not st[i]):
            primes[cnt] = i
            cnt += 1
        for j in range(i, x+1, i):
            st[j] = True
    print(cnt)
    
main()
```
